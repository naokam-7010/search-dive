# 機能仕様

## 1. ホーム（Dashboard）

**ルートID:** `dashboard`
**ファイル:** `src/views/Dashboard.tsx`

### 機能一覧

#### 1-1. クイック取得導線
- 「今すぐ取得する」ボタンをクリックするとクイック取得画面へ遷移
- 検索エンジン選択（Google / Yahoo / Bing）
- デバイス選択（PC / SP）
- 取得可能データの概要表示（ボリューム推移・SERPフィーチャー・推定流入ドメイン）

#### 1-2. Quick Stats パネル
- 今月の総取得キーワード数（例: 24,802）
- 保存済みキーワード数
- アクティブプロジェクト数
- 今月のエクスポート回数
- 「詳細レポートを見る」ボタン

#### 1-3. 直近の実行ジョブ一覧
- ジョブ名・対象件数・検索エンジン・実行日時の表示
- ステータスに応じたアイコン表示
  - `completed` → チェックアイコン（緑）
  - `processing` → 回転アニメーション（青）
  - `failed` → アラートアイコン（赤）
- ホバー時に「結果を見る」ボタンを表示

#### 1-4. 最近のプロジェクト一覧
- プロジェクト名・ドメイン・キーワード数・平均順位を表示
- カードクリックでプロジェクト詳細画面へ遷移

## 2. クイック取得（QuickAcquisition）

**ルートID:** `quick`
**ファイル:** `src/views/QuickAcquisition.tsx`

### 機能一覧

#### 2-1. 入力タブ切り替え
3種類の入力方式を切り替え可能。

**キーワード入力タブ**
- テキストエリアに1行1キーワード形式で貼り付け

**URL入力タブ**
- URL を入力して「候補を生成」ボタンをクリック
- Gemini API によるコンテンツ解析でキーワード候補を自動抽出・分類
- 生成されたキーワード候補はカテゴリ別（指名・比較・課題解決・情報収集）に表示
- チェックボックスで個別選択・全選択が可能
- 各カテゴリ内のキーワードは「編集」ボタンで修正可能

**ファイルアップロードタブ**
- CSV または TXT ファイルをドラッグ&ドロップ（最大 10,000行）
- 文字コード選択（UTF-8 / Shift-JIS）
- 区切り文字選択（カンマ / タブ）

#### 2-2. 取得設定
- 検索エンジン選択（Google / Yahoo / Bing）
- デバイス選択（PC / SP）
- 取得項目の選択（チェックボックス）
  - 検索ボリューム
  - 順位
  - LP / Title
  - SERP 機能
  - 流入ドメイン
  - 5年推移

#### 2-3. 取得実行・進捗表示
「取得を実行する」ボタンをクリックすると以下のステップで進捗を表示。

1. キーワードを整理中... (10%)
2. カテゴリを自動分類中... (30%)
3. 検索ボリュームを集計中... (60%)
4. 順位・LP情報を取得中... (85%)
5. 結果を整形しています... (100%)

プログレスバーとテキストで進捗を可視化。

#### 2-4. 結果表示

**サマリーカード（6枚）**
- 対象キーワード数
- 平均検索ボリューム
- ボリューム有り率
- 30位以内率
- 検出LP数
- 自動分類カテゴリ数

**フィルターバー**
- キーワード・URL・タイトルによるテキスト絞り込み
- カテゴリフィルター
- 順位範囲フィルター
- ボリュームフィルター
- 列表示切り替え
- CSV出力ボタン

**データテーブル**
- 横スクロール対応（min-width: 1200px）
- カテゴリ・検索クエリ列は sticky（スクロール固定）
- 表示カラム: カテゴリ / 検索クエリ / 12ヶ月平均 / 直近月 / 推移（スパークライン） / 順位 / SERP機能 / LP・タイトル
- 順位のカラーコーディング
  - 1〜3位: 緑背景（`bg-emerald-500`）
  - 4〜10位: 薄緑（`bg-emerald-100`）
  - 11位以下・圏外: グレー（`bg-zinc-100`）
- 行クリックでキーワード詳細ドロワーを開く
- ページネーション

#### 2-5. キーワード詳細ドロワー
画面右側からスライドインするドロワー。

- 現在の順位（前回比の矢印アイコン付き）
- 検索ボリューム（直近月・12ヶ月平均）
- SERP 機能一覧
- 月次検索ボリューム推移グラフ（過去12ヶ月、エリアチャート）
- LP・タイトル情報
- 想定流入ドメインシェア TOP3（プログレスバー）
- 「詳細を CSV 出力」ボタン
- 「プロジェクトに保存」ボタン

#### 2-6. エクスポートモーダル
- 出力形式選択（CSV / Excel）
- 出力範囲選択（全件 / フィルタ結果のみ / 選択した行のみ）
- ダウンロード実行後、エクスポート履歴画面からダウンロード可能

#### 2-7. プロジェクト保存モーダル
- 保存先の選択（既存プロジェクトに追加 / 新しいプロジェクトを作成）
- プロジェクト名の選択

## 4. プロジェクト詳細（ProjectDetail）

**ルートID:** `project-detail`
**ファイル:** `src/views/ProjectDetail.tsx`

### 機能一覧

#### 4-1. ヘッダー
- 「← 戻る」ボタンでプロジェクト一覧へ戻る
- プロジェクト名・ドメイン・最終更新日表示
- 「最新データに更新」ボタン（リフレッシュアニメーション付き）
- 「レポート出力」ボタン

#### 4-2. KPI サマリー（5指標）
- 平均順位（前回比付き）
- 10位以内率
- 30位以内率
- 総インプレッション
- 対象キーワード数

#### 4-3. グラフセクション

**順位・インプレッション推移グラフ（左）**
- 期間切り替え（7日間 / 30日間）
- エリアチャート（インプレッション）+ 折れ線グラフ（平均順位）の二軸表示
- Y軸左: 平均順位（逆順）/ Y軸右: インプレッション

**カテゴリ別パフォーマンスグラフ（右）**
- 横棒グラフ（カテゴリ別の総インプレッション）
- カテゴリ: 指名 / 比較検討 / 情報収集 / 課題解決

#### 4-4. キーワード一覧テーブル
- テキスト検索・フィルター
- 表示カラム: キーワード / カテゴリ / ボリューム / 順位 / 前回比 / LP

## 6. エクスポート履歴（ExportHistory）

**ルートID:** `history`
**ファイル:** `src/views/ExportHistory.tsx`

### 機能一覧

- ファイル名によるテキスト検索
- フィルター（出力形式・ステータス）
- エクスポート履歴リスト
  - ファイル名・出力形式バッジ（CSV / xlsx）
  - 件数・適用条件・実行日時
  - ステータス表示（Completed / Failed）
  - 「再ダウンロード」ボタン
  - メニューボタン
- 履歴なし時のエンプティーステート表示
- 保存期間: 30日間
